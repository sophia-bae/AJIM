Set Up Search Committees

By default, applications to all positions can be accessed by all faculty members in your group unless they are restricted by creating committees. The following is a list of all the current positions with their committee settings. Click on the [*tag*] links below to create/update a search committee for each position.

If you really need to further limit some faculty's permissions or grant additional access outside committees, or limit a group manager to some positions, you can do that for one faculty or multiple faculty. You can also list the access restrictions on applications, or update the ad hoc search committee view for each faculty, only if necessary. You should rarely need these links.
